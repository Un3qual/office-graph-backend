# API Migration Ledger

This ledger covers hand-written GraphQL and JSON API surfaces that remain live
while Office Graph migrates toward AshGraphql, AshJsonApi, and narrow custom
command or projection exceptions.

| Surface ID | Owner | Capability | Current surface | Exception class | Reason | Replacement target | Safety/parity tests | Retirement condition |

| `graphql.query.operator_inbox` | `OfficeGraph.Projections` | Operator workflow projection | `OfficeGraphWeb.GraphQL.OperatorWorkflow.Queries.operator_inbox` | projection exception | Mixed policy-filtered projection with actionability, watermarks, and product status. | GraphQL projection contract backed by `OfficeGraph.Projections.operator_inbox/1`. | `test/office_graph_web/operator_workflow_api_test.exs`. | Retire manual field only when generated reads or a modular GraphQL projection field replaces it with parity. |
