# All Runs Product Surface Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use
> `superpowers:subagent-driven-development` (recommended) or
> `superpowers:executing-plans` to implement this plan task-by-task. Steps use
> checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a bootstrap-only `/runs` workspace that pages authorized runs,
shows detailed run and activity state, and deep-links into the existing packet
and operator command surfaces.

**Architecture:** Add one focused, actor-authorized run-index projection and a
Relay connection for list data. Reuse the existing `operatorRunState` and
activity connection for detail; keep the new route read-only and extend the
packet route only enough to honor an explicit packet URL selection.

**Tech Stack:** Elixir 1.20, Phoenix, Ash, AshGraphql, PostgreSQL, Absinthe
Relay, React Router, React 19, Relay, TypeScript, Vitest, Nix, OpenSpec.

## Global Constraints

- OpenSpec is the development workflow only; it is not an Office Graph product
  feature or runtime dependency.
- Identity and governance administration remains deferred. This batch uses the
  existing bootstrap session and does not claim non-bootstrap feature
  completeness.
- The `/runs` route is read-only. Existing operator and packet routes remain
  the only mutation owners.
- Reuse `operatorRunState` and its activity connection for detail. Do not add a
  second run-detail or timeline model.
- Add no database migration, compatibility route, Tailwind dependency,
  provider/model call, direct external write, or hidden session fallback.
- The run index must be organization/workspace scoped, actor authorized,
  newest-first, keyset paginated, and constant-query as result size grows.
- Prefer actor-authorized Ash reads and batched relationship loading. Do not add
  a raw-SQL architecture exception merely to implement the list.
- Use behavior tests; do not add source-string assertions for runtime behavior.
- Generated Relay artifacts are regenerated by project tooling and are never
  hand edited.
- Run all project and OpenSpec commands through the repository Nix flake.
